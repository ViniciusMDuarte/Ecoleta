## Para iniciar

```bash

    # clone repository
    $ git clone https://github.com/ViniciusMDuarte/NextLevelWeek_Ecoleta.git

    # Enter directory
    $ cd NextLevelWeek_Ecoleta

    # Install package
    $ npm install
    
    # Start the project
    $ npm start

    # View browser
    $ http://localhost:3000/
```
